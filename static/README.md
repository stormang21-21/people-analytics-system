# Static Assets

Place your custom logo here as `logo.png`

## Logo Requirements:
- Format: PNG with transparency
- Size: 40x40 pixels (will be displayed at 40x40)
- Recommended: Square aspect ratio

## To use custom logo:
1. Save your logo as `logo.png` in this folder
2. Restart the app
3. The logo will automatically replace the emoji icon
