# People Analytics Workspace

This is the dedicated workspace for the People Analytics System project.

## Project Location
`~/workspace/people-analytics/`

## Quick Start

```bash
# Run the server
cd ~/workspace/people-analytics
./run_server.sh
```

Then open http://localhost:5051 in your browser.

## Features
- Real-time person detection & tracking
- Age/gender demographics
- Multi-camera support
- 2K video with AI-optimized processing
- Dark mode UI
